# Flows Audio

Code-native synthesis tools and experiments.

Flows is a project about treating source code as the live surface of a musical
instrument: readable, editable, inspectable, and directly connected to sound.

## Documentation

- [Language Reference](/reference/) — the `.flow` language as it ships today:
  the [Producer-mode standard library](/reference/producer) (oscillators,
  filters, envelopes, effects) and the [Developer-mode primitives](/reference/developer)
  it is built from.
